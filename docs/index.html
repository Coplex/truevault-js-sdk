TrueVaultClient

A client for the TrueVault HTTP API.

If you already have an API key or access token, use the constructor. If you have a username and password, see login().

new TrueVaultClient(apiKeyOrAccessToken: any)
Parameters
apiKeyOrAccessToken (any)

▸ login(accountId, username, password, mfaCode?)

Useful when you want to create a client starting from a user's username and password as opposed to an API key or access token. See https://docs.truevault.com/authentication#login-a-user.

login(accountId: string, username: string, password: string, mfaCode: string?): Promise<TrueVaultClient>
Parameters
accountId (string) account id that the user belongs to.
username (string) user's username.
password (string) user's password.
mfaCode (string?) current MFA code, if user has MFA configured.
